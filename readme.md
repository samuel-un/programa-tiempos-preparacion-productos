# Product Preparation Time Calculator

This Java application calculates the total preparation time for a product, allowing users to input each process and its duration.

## Project Structure

-   `src/model`: Data models (Product, Process)
-   `src/controller`: Business logic
-   `src/view`: Graphical interface (to be implemented)

## How to Run

Compile and run `Main.java` from the `src` folder.

## Branches

-   `feature/time-calculation`: Business logic
-   `feature/gui`: Graphical user interface
